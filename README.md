1. **三级代理返利体系**：高级会员299马币/直推75马币、中级199马币/直推35马币、普通免费
2. **TikTok达人+高校课程**：培养"国际机票+TikTok电商"复合人才
3. **即时通讯机器人工具**：Telegram/WhatsApp/WeChat机器人实时报价支付
