Singapore · 2026

执行摘要

TripNexus Global 是由今通与 Flowin 联合在新加坡设立的海外新品牌，面向东南亚市场，打造集 B2B Travel Marketplace、Travel SaaS、Payment Clearing 于一体的航旅 B2B 基础设施平台。

市场机会

东南亚在线旅游 GMV 预计 2025 年突破 USD 51B。市场持续恢复，在线化渗透率快速提升。这是一个高度碎片化的市场——旅行社、票代、TMC 分散经营，缺乏统一的交易与结算平台。

机会不是再做一个 OTA，而是构建旅行分销背后的 B2B 基础设施。

核心痛点

代理商

  • 缺乏稳定的全球票源接入渠道
  • 手工开票、退改操作效率低下
  • 无统一订单与财务管理系统
  • 多币种收付款结算困难重重
  • 每日平均浪费 2-3 小时在手工对账

TMC 差旅管理公司

  • 差旅合规管控工具缺失
  • 报告与对账功能薄弱
  • 与企业 ERP 系统集成复杂
  • 实时费用追踪能力不足
  • 企业差旅合规风险每年损失超 15%

供应商

  • B2B 分销渠道高度碎片化
  • 佣金结算周期长达 30-90 天
  • 缺乏实时库存与销售数据
  • 无统一的 B2B 直连 API 通道
  • 过度依赖第三方 GDS 中间商

为什么现在做

四大结构性驱动因素同向共振：

  • 旅游强劲复苏 — 东南亚旅游人次 2024 年已恢复至 2019 年的 112%
  • 数字化转型加速 — 东南亚 SMB 数字化采购率年增 38%
  • 可对标成熟市场 — Mondee、Navan 等模型已在美国验证，东南亚无对标玩家
  • 支付复杂性激增 — 跨境 B2B 支付规模预计 2026 年达 $42B

解决方案 — 三层架构

Layer 1 · B2B Travel Marketplace 交易平台

GDS 接入 (Amadeus/Sabre) · 航司直连 · 酒店床位 · 多渠道分销 API

Layer 2 · Travel SaaS 旅行企业管理系统

订单管理 · 财务管理 · 差旅合规 · 报表分析 · 白标部署

Layer 3 · Payment Clearing 多币种支付清算

多币种钱包 (SGD/MYR/THB/USD) · VCC 虚拟卡 · 自动清算 · 信用额度管理

Future Layer · VCC + AI 智能层

虚拟信用卡全自动发卡 · AI 定价引擎 · 智能风控 · 自然语言预订助手 (Coming 2025)

商业模式 — 四大收入池

1. 交易收入 — 机票/酒店/保险佣金 + 价差，佣金率 2-5%

2. SaaS 订阅收入 — USD 299-999/月/户，Year 3 目标 500 户 → ARR $3.5M

3. 支付手续费 — 换汇手续费 + VCC 服务费，0.5-1.5% 费率

4. 金融服务 — 信用额度利息 + 风险服务费，年化 8-15%

Year 3 总收入目标：USD 8M+

对标 Mondee

Mondee (NASDAQ: MOND) 作为美国 B2B 机票分销平台，上市估值 $1.2B。TripNexus 在东南亚市场对标 Mondee，但核心差异在于自建多币种清算 + VCC 体系——东南亚多货币环境（6+ 币种）、跨境结算复杂度远超美国单一市场，这是 TripNexus 的核心护城河。

东南亚进入策略

  • Phase 1 (0-6月) — 新加坡：总部建立、MVP 开发、签约首批 20 家代理商
  • Phase 2 (6-12月) — 马来西亚：本地化、代理商社群营销、GMV 突破 $5M
  • Phase 3 (12-18月) — 泰国：入境旅游供应商对接、TMC 客户开发
  • Phase 4 (18-24月) — 印尼/越南：Pre-A 轮 $5M、GMV 目标 $80M

合资结构

今通 × Flowin 联合在新加坡注册成立合资公司。今通贡献技术资源（GDS 直连、票务系统、研发团队），Flowin 贡献合规与市场资源（MAS 牌照、东南亚渠道、本地关系网）。

技术架构

底层系统 IP 归各自所有，定制开发版本归 TripNexus Global。今通的票务系统和票源接口、蜂象云的 SaaS 和财务系统、Flowin 的支付清算系统、AeroCredix 的 VCC 风控模型分层集成，上层为 TripNexus 海外定制系统及品牌资产。
