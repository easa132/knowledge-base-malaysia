## 二、Traveloka for Business 具体入驻条件

### 2.1 Traveloka for Business 简介
Traveloka 的企业商旅平台，面向旅行社/企业客户提供机票+酒店+用车B2B分销。

### 2.2 入驻基本条件

| 条件类别 | 具体要求 |
|---------|---------|
| **公司资质** | 必须是合法注册的公司（SSM注册），经营范围含旅游业 |
| **注册资本** | 无明确最低要求，建议至少 RM 10万 |
| **行业许可** | 需要持有 MOTAC（马来西亚旅游部）旅行社执照 |
| **从业经验** | 至少 6-12 个月旅游行业运营记录 |
| **保证金/押金** | RM 5,000 - 20,000（视信用评级而定） |
| **最低交易量** | 首3个月月均至少 RM 50,000 交易额 |
| **团队要求** | 至少2名客服人员（马来语+英语） |

### 2.3 入驻流程

```
1. 提交申请 → travelokabusiness.com/apply
2. 提交公司文件（SSM、MOTAC、银行账单）
3. 背景审核（1-2周）
4. 签署合作协议
5. 系统接入（API或Web Portal）
6. 缴纳保证金
7. 正式上线
```

### 2.4 Traveloka for Business 定价/返佣

| 项目 | 详情 |
|------|------|
| **入驻费** | 免费 |
| **年费** | 无 |
| **机票佣金** | 1-3%（视航司和票种而定） |
| **酒店佣金** | 8-12% |
| **促销返点** | 达到月交易额目标额外 0.5-1% |
| **支付手续费** | 1.5-2.5%（由代理承担） |
| **结算周期** | T+7 至 T+30（视信用额度） |

## 三、Tripfez B2B 平台详情

### 3.1 Tripfez 简介
马来西亚本土穆斯林友好旅游B2B平台，专注做东南亚+中东市场。

### 3.2 Tripfez API 功能

| API模块 | 功能说明 |
|--------|---------|
| **Flight API** | 机票搜索、价格、预订、出票、退改 |
| **Hotel API** | 酒店搜索、预订（特别是穆斯林友好酒店） |
| **Umrah API** | 副朝套餐搜索预订 |
| **Package API** | 旅游套餐集成 |
| **Payment API** | 支付集成、退款处理 |
| **Booking Management** | 订单查询、取消、修改 |

### 3.3 Tripfez API 技术参数

| 项目 | 内容 |
|------|------|
| **接口协议** | RESTful JSON |
| **认证方式** | API Key + Secret |
| **请求格式** | JSON |
| **响应格式** | JSON |
| **沙箱环境** | ✅ 提供 Sandbox |
| **速率限制** | 30 req/min（基础） / 100 req/min（高级） |
| **SLA** | 99.5% uptime |
| **文档** | 入驻后提供 API Documentation |

### 3.4 Tripfez B2B 合作条件

| 项目 | 详情 |
|------|------|
| **入驻费** | 免费 |
| **月费** | 无（基础版）/ RM 500（高级版带API） |

## 七、关键洞察与战略建议

### 7.1 市场空白机会

| 空白领域 | 现状 | 今通的切入点 |
|---------|------|------------|
| **本地GDS + 中文支持** | Kiu System无中文，Atlantic Link老旧 | 开发有中文界面的平台 |
| **小额快捷结算** | 大部分平台T+14以上 | 提供T+0/T+1结算（收手续费） |
| **马来西亚-中国专线** | 无专门平台 | 深耕中马航线B2B |
| **穆斯林旅游+机票组合** | Tripfez酒店为主，机票弱 | 机票+酒店+朝觐套餐一体化 |
| **廉价航司API聚合** | AirAsia B2B门槛高 | 聚合AirAsia/MyAirline/Batik Air |

### 7.2 支付策略建议

```
【短期】接入 iPay88 + FPX + 银行转账（最低成本覆盖主流支付）
【中期】开通 DuitNow + 虚拟账户（提高大代理效率）
【长期】自建钱包体系（沉淀资金 + 加速结算）
```

### 7.3 返佣策略建议

| 代理等级 | 月交易额 | 建议返佣 | 额外激励 |
|---------|:-------:|:-------:|---------|

| 银牌 | RM 50-200k | 1.5-2.0% | 季度返点 0.3% |
